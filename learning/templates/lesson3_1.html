  • Point
  • 教學程式碼
  1. Bootstrap是熱門HTML CSS JS框架，使用者無需撰寫CSS或JS程式碼就能開發響應式網頁，而且CSS或JS之設計皆在HTML文本中完成。
  2. 至Bootstrap官網查閱Documentation，並取得添加套件程式碼。
  3. 主要檔案如下 :
    I ) Bootstrap核心CSS檔案 : bootstrap.min.css

    II ) 將網頁寬度指定為行動裝置的螢幕寬度，縮放比為1:1，當有元素超過螢幕寬度時，shrink-to-fit=no能夠讓網頁不被縮放以符合螢幕寬度
          < meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no" >
    III ) jQuery核心JavaScript檔案 : jQuery-XXX.min.js(XXX為版本)
    IV ) Popper.js : popper.min.js
    V ) Bootstrap核心JavaScript檔案 : bootstrap.min.js
